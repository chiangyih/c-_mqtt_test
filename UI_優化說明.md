## ?? 使用流程

1. 啟動程式 → 看到「● 未連線」灰色狀態
2. 輸入 Server IP → 點擊綠色「連線」按鈕
3. 連線中 → 狀態變為「● 連線中...」橘色
4. 連線成功 → 狀態變為「● 已連線」綠色
5. 輸入主題 → 點擊藍色「訂閱」按鈕
6. 在訊息區域看到接收的訊息
7. 點擊紅色「斷線」按鈕 → 確認後斷線
8. 狀態回到「● 未連線」灰色

## ?? 設計原則

1. **視覺層次**: 使用 GroupBox 清晰分組
2. **色彩語義**: 綠色/紅色/藍色代表不同動作類型
3. **即時反饋**: 狀態指示器即時更新
4. **易用性**: 輸入提示、標籤說明完整
5. **一致性**: 統一的字體、間距、配色
